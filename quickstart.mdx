title: "Quickstart"
description: "Stream live video from your Android camera to an RTMP, RTSP, or SRT server in minutes."
---

<Note>
  `GenericStream` requires Android API 21+. If you need to support API 16–20, see the [old API example](https://github.com/pedroSG94/RootEncoder/tree/master/app/src/main/java/com/pedro/streamer/oldapi) in the repository. See [Installation](/installation) for full dependency setup.
</Note>

<Steps>

<Step title="Add the JitPack repository">

In your project-level `build.gradle` (or `settings.gradle` if you use `dependencyResolutionManagement`), add the JitPack repository:

```groovy
allprojects {
  repositories {
    maven { url 'https://jitpack.io' }
  }
}
```

</Step>

<Step title="Add the library dependency">

In your app-level `build.gradle`, add the `library` module:

```groovy
dependencies {
  implementation "com.github.pedroSG94.RootEncoder:library:2.7.2"
}
```

</Step>

<Step title="Add permissions to AndroidManifest.xml">

Open your `AndroidManifest.xml` and add the following permissions:

```xml
<uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.RECORD_AUDIO" />
<uses-permission android:name="android.permission.CAMERA" />
```

</Step>

<Step title="Implement ConnectChecker">

`GenericStream` requires a `ConnectChecker` to receive connection lifecycle callbacks. Implement it in your `Activity` or `Fragment`:

```kotlin
import com.pedro.common.ConnectChecker

class MainActivity : AppCompatActivity(), ConnectChecker {

  override fun onConnectionStarted(url: String) {
    // Called when the connection attempt begins
  }

  override fun onConnectionSuccess() {
    // Called when the stream is connected and running
  }

  override fun onConnectionFailed(reason: String) {
    // Called when the connection fails — handle retries or show an error
  }

  override fun onDisconnect() {
    // Called when the stream disconnects
  }

  override fun onAuthError() {
    // Called when server authentication fails
  }

  override fun onAuthSuccess() {
    // Called when server authentication succeeds
  }
}
```

</Step>

<Step title="Create a GenericStream and start preview">

Add a `SurfaceView` to your layout:

```xml
<SurfaceView
  android:id="@+id/surfaceView"
  android:layout_width="match_parent"
  android:layout_height="match_parent" />
```

Then create a `GenericStream` and start the camera preview when the surface is ready:

```kotlin
import com.pedro.library.generic.GenericStream

val genericStream = GenericStream(this, this).apply {
  getGlInterface().autoHandleOrientation = true
}

surfaceView.holder.addCallback(object : SurfaceHolder.Callback {
  override fun surfaceCreated(holder: SurfaceHolder) {
    if (!genericStream.isOnPreview) genericStream.startPreview(surfaceView)
  }

  override fun surfaceChanged(holder: SurfaceHolder, format: Int, width: Int, height: Int) {
    genericStream.getGlInterface().setPreviewResolution(width, height)
  }

  override fun surfaceDestroyed(holder: SurfaceHolder) {
    if (genericStream.isOnPreview) genericStream.stopPreview()
  }
})
```

</Step>

<Step title="Prepare video and audio encoders">

Call `prepareVideo()` and `prepareAudio()` before starting a stream. Both return `true` on success.

```kotlin
val prepared = genericStream.prepareVideo(
  width = 640,
  height = 480,
  bitrate = 1_200_000  // 1.2 Mbps
) && genericStream.prepareAudio(
  sampleRate = 32000,
  isStereo = true,
  bitrate = 128_000  // 128 Kbps
)

if (!prepared) {
  // Handle configuration failure
}
```

</Step>

<Step title="Start and stop the stream">

Call `startStream()` with your server URL. `GenericStream` detects the protocol automatically from the URL scheme (`rtmp://`, `rtsp://`, `srt://`, or `udp://`).

```kotlin
// Start streaming
genericStream.startStream("rtmp://your-server/live/stream-key")

// Stop streaming
genericStream.stopStream()
```

</Step>

</Steps>

## Complete example

The following is a self-contained `Fragment` showing the full streaming flow, based on the [rotation example](https://github.com/pedroSG94/RootEncoder/tree/master/app/src/main/java/com/pedro/streamer/rotation) from the RootEncoder repository.

```kotlin
import android.os.Build
import android.os.Bundle
import android.view.LayoutInflater
import android.view.SurfaceHolder
import android.view.SurfaceView
import android.view.View
import android.view.ViewGroup
import androidx.annotation.RequiresApi
import androidx.fragment.app.Fragment
import com.pedro.common.ConnectChecker
import com.pedro.library.generic.GenericStream

@RequiresApi(Build.VERSION_CODES.LOLLIPOP)
class StreamFragment : Fragment(), ConnectChecker {

  private lateinit var surfaceView: SurfaceView

  val genericStream: GenericStream by lazy {
    GenericStream(requireContext(), this).apply {
      getGlInterface().autoHandleOrientation = true
    }
  }

  override fun onCreateView(
    inflater: LayoutInflater, container: ViewGroup?, savedInstanceState: Bundle?
  ): View? {
    val view = inflater.inflate(R.layout.fragment_stream, container, false)
    surfaceView = view.findViewById(R.id.surfaceView)

    surfaceView.holder.addCallback(object : SurfaceHolder.Callback {
      override fun surfaceCreated(holder: SurfaceHolder) {
        if (!genericStream.isOnPreview) genericStream.startPreview(surfaceView)
      }
      override fun surfaceChanged(holder: SurfaceHolder, format: Int, width: Int, height: Int) {
        genericStream.getGlInterface().setPreviewResolution(width, height)
      }
      override fun surfaceDestroyed(holder: SurfaceHolder) {
        if (genericStream.isOnPreview) genericStream.stopPreview()
      }
    })

    return view
  }

  override fun onCreate(savedInstanceState: Bundle?) {
    super.onCreate(savedInstanceState)
    val prepared = genericStream.prepareVideo(640, 480, 1_200_000)
        && genericStream.prepareAudio(32000, true, 128_000)
    if (!prepared) {
      activity?.finish()
    }
  }

  override fun onDestroy() {
    super.onDestroy()
    genericStream.release()
  }

  fun startStream(url: String) {
    genericStream.startStream(url)
  }

  fun stopStream() {
    genericStream.stopStream()
  }

  // ConnectChecker callbacks
  override fun onConnectionStarted(url: String) {}
  override fun onConnectionSuccess() {}
  override fun onConnectionFailed(reason: String) {
    genericStream.stopStream()
  }
  override fun onDisconnect() {}
  override fun onAuthError() {}
  override fun onAuthSuccess() {}
}
```
