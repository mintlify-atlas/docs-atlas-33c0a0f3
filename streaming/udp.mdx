<Warning>
  UDP streaming support in RootEncoder is in beta. The API is functional but may change in future releases.
</Warning>

`UdpStream` transmits MPEG-TS packets over UDP. Unlike RTMP or RTSP there is no connection handshake, so the stream begins sending immediately. UDP supports three addressing modes:

| Mode | Address range | Example URL |
|------|--------------|-------------|
| Unicast | Standard IP address | `udp://192.168.1.100:1234` |
| Multicast | 224.0.0.0 – 239.255.255.255 | `udp://224.0.0.1:1234` |
| Broadcast | 255.255.255.255 | `udp://255.255.255.255:1234` |

## Basic usage

<Steps>
  <Step title="Implement ConnectChecker">
    ```kotlin
    class MainActivity : AppCompatActivity(), ConnectChecker {

      override fun onConnectionStarted(url: String) {}
      override fun onConnectionSuccess() { /* sending packets */ }
      override fun onConnectionFailed(reason: String) { /* handle failure */ }
      override fun onDisconnect() {}
      override fun onAuthError() {}
      override fun onAuthSuccess() {}
    }
    ```
  </Step>

  <Step title="Create UdpStream">
    The default constructor uses `Camera2Source` and `MicrophoneSource`.

    ```kotlin
    val udpStream = UdpStream(context, connectChecker)
    ```

    Supply custom sources via the full constructor:

    ```kotlin
    val udpStream = UdpStream(
      context = this,
      connectChecker = this,
      videoSource = Camera2Source(context),
      audioSource = MicrophoneSource()
    )
    ```
  </Step>

  <Step title="Prepare video and audio">
    ```kotlin
    val prepared = udpStream.prepareVideo(
      width = 1280,
      height = 720,
      bitrate = 1_200_000
    ) && udpStream.prepareAudio(
      sampleRate = 44100,
      isStereo = true,
      bitrate = 128_000
    )
    ```
  </Step>

  <Step title="Start and stop the stream">
    ```kotlin
    // Unicast
    udpStream.startStream("udp://192.168.1.100:1234")

    // Multicast
    udpStream.startStream("udp://224.0.0.1:1234")

    // Broadcast
    udpStream.startStream("udp://255.255.255.255:1234")

    // Stop
    udpStream.stopStream()
    ```
  </Step>
</Steps>

<Note>
  UDP has no built-in retransmission. Packets lost in transit are not recovered. Use SRT if your network is unreliable and you need delivery guarantees.
</Note>

## Retry on failure

```kotlin
override fun onConnectionFailed(reason: String) {
  val retrying = udpStream.getStreamClient().reTry(5000L, reason)
  if (!retrying) {
    udpStream.stopStream()
  }
}
```

Set the maximum number of retry attempts:

```kotlin
udpStream.getStreamClient().setReTries(5)
```

## Stream delay

Buffer outgoing packets for a fixed delay before transmission. Useful when you need the sender and receiver clocks to stay aligned:

```kotlin
// Delay output by 2000 milliseconds
udpStream.getStreamClient().setDelay(2000L)
```

## Cache management

```kotlin
val client = udpStream.getStreamClient()

// Check for congestion (default 20% threshold)
val congested = client.hasCongestion()

// Resize the packet cache in bytes
client.resizeCache(32 * 1024 * 1024)

// Discard all queued packets
client.clearCache()
```

## Statistics

```kotlin
val client = udpStream.getStreamClient()

val sentVideo = client.getSentVideoFrames()
val sentAudio = client.getSentAudioFrames()
val dropped   = client.getDroppedVideoFrames()
val bytes     = client.getBytesSend()
```
